# 📰 SpeedNews

> 다양한 언론사의 뉴스를 한눈에 모아보는 **실시간 뉴스 피드 웹 서비스**  
> Spring Boot + React + PostgreSQL 기반의 개인 프로젝트입니다.

<br>

## 📖 프로젝트 소개

여러 언론사의 뉴스를 각각 방문해서 보는 게 불편하다는 이야기를 듣고,  
‘한눈에 다양한 뉴스를 볼 수 있는 간단한 웹 서비스’를 만들어보고자 시작했습니다.  

**SpeedNews**는 RSS 피드를 자동으로 수집해 사용자에게 실시간으로 보여주는 뉴스 피드 서비스입니다.  
초기 목표(MVP)는 ‘RSS 수집 및 화면 출력’이었고,  
현재는 **회원가입·로그인 기능**을 추가 개발 중입니다.  
이후에는 **즐겨찾기, 언론사 확장, 사용자 맞춤 뉴스** 등의 기능을 점차 도입할 예정입니다.

<br>

## 🧰 Tech Stack

### 🖥️ Frontend
| 기술 | 사용 목적 |
|------|------------|
| **React (TypeScript)** | 간단한 UI 구성 및 컴포넌트 관리 |
| **Vite** | 빠른 빌드 환경 및 로컬 개발 서버 |
| **Axios** | 백엔드 API 요청 처리 |
| **(기본 CSS)** | 간단한 스타일 적용 |

### ⚙️ Backend
| 기술 | 사용 목적 |
|------|------------|
| **Spring Boot (3.x)** | 백엔드 서버 및 REST API 구현 |
| **Spring Security + JWT** | 로그인 세션 대신 토큰 기반 인증 |
| **Email Verification** | 회원가입 시 이메일 인증 기능 |
| **JPA (Hibernate)** | 데이터베이스 접근 및 객체 매핑 |
| **Lombok** | 반복되는 getter/setter 등 코드 간소화 |

### 🗄️ Database & Infra
| 기술 | 사용 목적 |
|------|------------|
| **PostgreSQL (Docker)** | 뉴스 및 사용자 데이터 저장 |
| **Flyway** | DB 스키마 버전 관리 |
| **Docker Compose** | DB와 서버 환경 통합 실행 |
| **Render / Netlify** | 서비스 배포 (백엔드 / 프론트엔드) |

<br>

## ✨ 주요 기능

### 🎯 MVP (최초 버전)
- 🔍 **RSS 피드 수집 및 화면 출력**  
  → 다양한 언론사의 RSS 데이터를 수집해 웹에 출력

### 🧩 현재 추가 중
- 🔑 회원가입 / 로그인 (JWT 인증)
- 📧 이메일 인증 기능

### 🔮 향후 계획
- ⭐ 즐겨찾기(북마크) 기능  
- 📰 언론사 확장  
- 💡 사용자 맞춤 뉴스 추천

## 💬 회고

SpeedNews는 처음으로 프론트엔드와 백엔드를 함께 구현하고 배포한 프로젝트입니다.
